The 5 HTML tags I learnt are:

Paragraph tag <p></p>:
This tag is used to write the paragraphs which means to sepreate the texts where and how you want.
Heading Tags <h1></h1>:
It is something which differentiates your other texts with the heading text. It can be used like h1, h2, h3, h4, h5, h6 , whichever size you want for your text heading.
Emphasis Tag <em></em>:
It is used to make the text look italic.
Image Tag <img>:
It is used to insert an image in the main page.
Title Tag <title></title>:
This tag is use to give a title to your webpage.
